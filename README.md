# 🍳 Cook Together, Grow Together 🌿
**Meal prep meets mindful parenting and multilingual learning.**

---

## ✨ The Vibe
"Cook Together, Grow Together" is a web-based interactive experience designed to turn the "chore" of meal prepping into a bonding activity. It bridges the gap between a parent's need to get dinner ready and a child's natural curiosity to learn and play.

### 🌿 Core Philosophy
* **Parent + Child:** Dual-zone instructions so everyone has a mission.
* **Language Immersion:** Lightweight learning in **English, Spanish, Simplified Chinese, and Traditional Chinese**.
* **Montessori-Inspired:** Focused on tactile engagement, counting, and visual association.

---

## 🚀 Features
- **Role Selection:** Toggle between "Parent" and "Little Chef" views.
- **Magic Words:** High-visibility vocabulary cards with **Pinyin** support for Chinese learners.
- **Celebration States:** Reward your Little Chef with digital "stickers" and positive reinforcement.
- **Mobile First:** Designed to look beautiful on a tablet or phone propped up on a kitchen counter.

---

## 🛠️ How to Run This App
This is a **Zero-Install** project. Because it is built as a single-file web app, you can run it anywhere:

1. **On the Web:** [CLICK HERE TO OPEN THE LIVE APP](https://catherine-fu88.github.io/Cook-and-Grow-Together/)
2. **Locally:** - Download the `index.html` file.
   - Double-click it to open in any modern web browser (Chrome, Safari, Firefox).
